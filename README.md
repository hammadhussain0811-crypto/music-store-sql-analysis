# Music Store SQL Data Analysis

## Project Overview

This project analyzes a **digital music store database (Chinook)** using SQL.
The goal is to explore customer behavior, sales trends, and music preferences by writing analytical SQL queries.

The project demonstrates the use of **SQL joins, aggregations, subqueries, and window functions** to extract meaningful insights from the database.

---

## Database

The project uses the **Chinook database**, a sample database that represents a digital music store.

The database includes information about:

* Artists
* Albums
* Tracks (songs)
* Customers
* Invoices
* Employees
* Playlists
* Genres

## SQL Analysis

The project includes three levels of SQL queries.

### Easy Queries

Basic SQL queries used to explore the dataset.

Examples:

* Identify the senior most employee.
* Find countries with the most invoices.
* Determine the top invoice values.

### Moderate Queries

Queries using joins and filtering.

Examples:

* Identify Rock music listeners.
* Find top rock artists.
* Identify tracks longer than average song length.



## Key SQL Concepts Used
